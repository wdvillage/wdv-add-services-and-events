== Description ==

The plugin "WDV - Add Services and Events" added custom post types "Services" and "Events" to WP theme "Aniro Hotel".

== Installation ==

1. Upload the plugin files to the /wp-content/plugins/wdv-add-services-and-events` directory, or install the plugin through the WordPress plugins screen directly.
2. In your WordPress administration, go to the Plugins page.
3. Activate the 'WDV - Add Services and Events' plugin through the 'Plugins' screen in WordPress. 
4. Now you can find custom post types 'Services' and 'Events' in admin panel and you can create new services and events.
